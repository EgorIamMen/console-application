# Console Application

Консольное приложение на **Python**, реализованное с разделением логики на слои
(UI / Application / Domain / Infrastructure).  
Проект предназначен для учебных и практических целей и демонстрирует
структурированный подход к разработке консольных программ.

---

## Стек технологий

- Python 3.10+
- Стандартная библиотека Python
- Консольный интерфейс (CLI)

---

## Структура проекта

```
.
├── main.py             # Точка входа в приложение
├── ui.py               # Консольный интерфейс (ввод/вывод, меню)
├── application.py      # Прикладная логика (use-cases)
├── domain.py           # Доменная модель и бизнес-правила
├── infrastructure.py  # Работа с внешними ресурсами
├── global_state.py    # Глобальное состояние приложения
└── __pycache__/        # Служебные файлы Python (генерируются автоматически)
```

---

## Архитектура

Проект разделён на логические слои:

- **UI** — отвечает за взаимодействие с пользователем
- **Application** — содержит сценарии использования и координирует работу логики
- **Domain** — описывает предметную область и бизнес-правила
- **Infrastructure** — работа с файлами и другими внешними ресурсами
- **Global State** — общее состояние, используемое между модулями

Такое разделение упрощает поддержку, расширение и тестирование приложения.

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/EgorIamMen/console-application.git
cd console-application
```

### 2. (Рекомендуется) Создание виртуального окружения

**Windows:**
```powershell
py -m venv .venv
.\.venv\Scripts\activate
```

**Linux / macOS:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Запуск приложения
```bash
python main.py
```

---

## Запуск в VS Code

1. Открой папку проекта в VS Code  
2. Выбери интерпретатор Python (желательно из `.venv`)  
3. Запусти `main.py` через Run / Debug  
   или через встроенный терминал:
```bash
python main.py
```
